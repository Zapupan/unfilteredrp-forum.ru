# Настройка сервера для MySQL базы данных

## У вас уже есть MySQL база данных на хостинге!

База данных: `u3372230_unfilteredrp-bd`

## Что нужно сделать:

### 1. Установить зависимости

```bash
npm install
```

Это установит `mysql2` для работы с MySQL.

### 2. Найти данные для подключения к базе данных

В панели управления хостингом найдите:
- **Хост (Host)**: обычно `localhost` или что-то типа `mysql.hosting.com`
- **Пользователь (User)**: имя пользователя БД (например, `u3372230_default`)
- **Пароль (Password)**: пароль от базы данных
- **База данных (Database)**: `u3372230_unfilteredrp-bd`

### 3. Настроить подключение

Откройте файл `server-mysql.js` и найдите строки:

```javascript
const dbConfig = {
    host: process.env.DB_HOST || 'localhost',
    user: process.env.DB_USER || 'u3372230_default', // ЗАМЕНИТЕ!
    password: process.env.DB_PASSWORD || '', // ЗАМЕНИТЕ!
    database: process.env.DB_NAME || 'u3372230_unfilteredrp-bd',
    // ...
};
```

**Замените:**
- `u3372230_default` на вашего пользователя БД
- `''` на ваш пароль БД
- `localhost` на хост (если не localhost)

### 4. Импортировать структуру базы данных

1. Откройте панель управления хостингом
2. Найдите вашу базу данных `u3372230_unfilteredrp-bd`
3. Нажмите **"Web интерфейс БД"** (phpMyAdmin)
4. Выберите базу данных слева
5. Перейдите на вкладку **"Импорт"**
6. Выберите файл `database_dump.sql`
7. Нажмите **"Вперед"**

### 5. Запустить сервер

```bash
node server-mysql.js
```

Или переименуйте файл:
```bash
mv server-mysql.js server.js
npm start
```

## Использование переменных окружения (безопаснее)

Вместо хранения пароля в коде, можно использовать переменные окружения:

Создайте файл `.env`:
```
DB_HOST=localhost
DB_USER=u3372230_default
DB_PASSWORD=ваш_пароль
DB_NAME=u3372230_unfilteredrp-bd
```

И установите пакет:
```bash
npm install dotenv
```

Добавьте в начало `server-mysql.js`:
```javascript
require('dotenv').config();
```

Если видите ошибку подключения - проверьте данные для подключения!

## Где найти данные для подключения?

В панели управления хостингом:
- Обычно в разделе **"Базы данных"** или **"MySQL"**
- Или в разделе **"Доступ к БД"**
- Может быть кнопка **"Параметры подключения"**
